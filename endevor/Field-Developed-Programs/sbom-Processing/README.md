# sbom-Processing

Watch this folder for examples of sbom processing in Endevor.
Expected considerations to include:

- An Exit-driven process to insert a Generated sbom into a package at the time of a package CAST action. The sbom will contain details for all the other elements in the package.
- Processor examples to validate the sbom element as a part of a package execution.
- Package shipping examples for validating a shipped sbom

The intent for these examples will be to provide configurations that may require a 1-time implementation, but then fit seamlessly into your existing structure, without requiring any additional manual steps.
